# ZipIt Logo Files - Exact Sizes Required

## 📏 Logo Size Requirements:

### Main Logo:
- **logo.svg** - Vector (scalable to any size)
- **logo-horizontal.png** - 400x100px (4:1 ratio)
- **logo-square.png** - 200x200px (1:1 ratio)

### PWA/Mobile Icons:
- **logo-192.png** - 192x192px (PWA standard)
- **logo-512.png** - 512x512px (PWA standard)
- **favicon.ico** - 32x32px (browser tab)
- **favicon-16.png** - 16x16px (small favicon)

### Dashboard Headers:
- **logo-nav.png** - 120x40px (navigation bar)
- **logo-footer.png** - 80x30px (footer)

### Social/Marketing:
- **logo-og.png** - 1200x630px (social media sharing)
- **logo-banner.png** - 800x200px (website banner)

## 📋 File Format Guidelines:
- **SVG**: Vector logo (preferred for web)
- **PNG**: Transparent background
- **ICO**: For favicon only
- **Quality**: High resolution, crisp edges
- **Colors**: Include both light and dark versions

## 🎨 Design Tips:
- Simple, recognizable at small sizes
- Works on both light and dark backgrounds
- Lightning bolt (⚡) theme preferred
- Professional, tech-focused appearance
